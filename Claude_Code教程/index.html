Claude Code

智能编程助手安装与配置完全指南

开始前的准备

本教程将引导您完成 Claude Code 的完整安装和配置过程。整个过程大约需要 15-20 分钟，您需要准备以下环境：

  • 具有管理员权限的计算机（Windows、macOS 或 Linux）
  • 稳定的网络连接
  • 基本的命令行操作经验
1

安装 Node.js 和 npm

Claude Code 依赖 Node.js 环境和 npm 包管理工具。这是使用 Claude Code 的必要前提条件。

步骤 1.1：检查当前环境

首先检查您的系统是否已经安装了 Node.js 和 npm。打开终端（Terminal 或 CMD），执行以下命令：

检查版本命令
node -v
npm -v

✅ 如果显示版本号（如 v18.0.0 或更高），说明已经安装，可以跳过此步骤。

步骤 1.2：安装 Node.js 和 npm

根据您的操作系统选择相应的安装方法：

Windows 和 macOS 用户

推荐使用官方安装包，这是最简单可靠的方式：

🔗 访问 Node.js 官网下载

选择 LTS（长期支持）版本并下载安装。npm 会随 Node.js 一起自动安装。

Linux 用户（Ubuntu/Debian）

Ubuntu/Debian 安装命令
# 添加 NodeSource 仓库
curl -fsSL https://deb.nodesource.com/setup_lts.x | sudo -E bash -

# 安装 Node.js
sudo apt-get install -y nodejs

验证安装

安装完成后，重新执行版本检查命令确认安装成功：

验证安装
node -v
npm -v
2

安装 Claude Code CLI

现在使用 npm 全局安装 Claude Code 命令行工具。这将使您能够在任何目录中使用 claude 命令。

步骤 2.1：执行安装命令

在终端中执行以下命令：

安装 Claude Code
npm install -g @anthropic-ai/claude-code

步骤 2.2：验证安装

安装完成后，验证 Claude Code 是否正确安装：

验证 Claude Code 安装
claude --version

✅ 如果显示版本信息，说明安装成功。您现在可以使用 claude 命令调用 Claude Code。

🔧 故障排除

• 如果遇到权限问题，请使用管理员权限运行终端

• 如果 claude 命令不可用，请重启终端或检查 PATH 环境变量

3

配置 Settings 文件

Claude Code 需要配置 API Key 和相关环境变量。

创建配置文件

在用户主目录创建配置文件：

  • macOS/Linux: ~/.claude/settings.json
  • Windows（WSL）: ~/.claude/settings.json

如果 .claude 文件夹不存在，请先创建。

编辑 settings.json 内容示例：

settings.json
{
  "env": {
    "ANTHROPIC_API_KEY": "your-api-key-here",
    "ANTHROPIC_BASE_URL": "https://anyrouter.top",
    "CLAUDE_CODE_MAX_OUTPUT_TOKENS": 64000,
    "CLAUDE_CODE_DISABLE_NONESSENTIAL_TRAFFIC": 1
  },
  "permissions": {
    "allow": [],
    "deny": []
  },
  "apiKeyHelper": "echo 'your-api-key-here'"
}
⚠️ 注意

• 将 "your-api-key-here" 替换为您实际的 API Key。

• 若使用 opus 模型，请将 CLAUDE_CODE_MAX_OUTPUT_TOKENS 改为 32000，且删除注释。

5

项目配置（可选）

建议在项目根目录创建 CLAUDE.md 文件，保存项目架构、编码规范和工作流程，帮助 Claude Code 记忆项目上下文。

6

开始使用 Claude Code

进入项目目录，执行：

启动 Claude Code
claude

即可启动 Claude Code，开始体验智能编码辅助。

附录：环境变量说明

变量名 说明
ANTHROPIC_API_KEY 您的 API 访问密钥
ANTHROPIC_BASE_URL API 服务基础 URL，默认 https://anyrouter.top
CLAUDE_CODE_MAX_OUTPUT_TOKENS 最大输出 token 数，opus 模型建议 32000，默认 64000
CLAUDE_CODE_DISABLE_NONESSENTIAL_TRAFFIC 禁止非必要流量发送，建议设置为 1
